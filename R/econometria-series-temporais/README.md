## 📌 Sobre

Repositório de scripts em **R Markdown** desenvolvidos ao longo da disciplina de Econometria II — Séries Temporais. Os arquivos cobrem aplicações práticas de modelagem econométrica em dados reais.

---

## 🗂️ Arquivos

| Arquivo | Descrição |
| :--- | :--- |
| `Câmbio_Won.Rmd` | Análise de séries temporais da taxa de câmbio Won coreano |
| `Etherium.Rmd` | Modelagem de séries do preço do Ethereum |
| `GARCH.Rmd` | Estimação de modelos GARCH para volatilidade |
| `Investimento.Rmd` | Análise de séries de investimento |
| `IPCA.Rmd` | Modelagem do índice de inflação IPCA |
| `Lista 4.Rmd` | Lista de exercícios — parte 1 |
| `Lista_4_parte_2.Rmd` | Lista de exercícios — parte 2 |
| `Petrôbras.Rmd` | Análise de séries temporais das ações da Petrobras |
